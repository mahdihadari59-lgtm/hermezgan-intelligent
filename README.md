## рҹҡҖ ШҙШұЩҲШ№ ШіШұЫҢШ№

### ЩҫЫҢШҙвҖҢЩҶЫҢШ§ШІЩҮШ§
- Python 3.9+
- Node.js 16+
- Git
- Docker & Docker Compose (Ш§Ш®ШӘЫҢШ§ШұЫҢ)

### ЩҶШөШЁ ЩҲ Ш§Ш¬ШұШ§

```bash
# Clone Repository
git clone https://github.com/mahdihadari59-lgtm/hermezgan-intelligent.git
cd hermezgan-intelligent

# Backend Setup
cd backend
python -m venv venv
source venv/bin/activate  # ЫҢШ§: venv\Scripts\activate (Windows)
pip install -r requirements.txt

# Database Setup
python scripts/seed_database.py

# Run Backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# Frontend Setup (ШҜШұ Terminal Ш¬ШҜЫҢШҜ)
cd ../frontend
npm install
npm start
```

### Docker Setup

```bash
docker-compose up -d
```

## рҹ”Ң API Endpoints

### Chat API
```
POST   /api/v1/chat/message      - Ш§ШұШіШ§Щ„ ЩҫЫҢШ§Щ…
GET    /api/v1/chat/history      - ШҜШұЫҢШ§ЩҒШӘ ШӘШ§ШұЫҢШ® ЪҜЩҒШӘЪҜЩҲ
```

### Knowledge Graph
```
GET    /api/v1/graph/entities    - ШҜШұЫҢШ§ЩҒШӘ Щ…ЩҲШ¬ЩҲШҜЫҢШӘвҖҢЩҮШ§
GET    /api/v1/graph/relations   - ШҜШұЫҢШ§ЩҒШӘ ШұЩҲШ§ШЁШ·
POST   /api/v1/graph/entity      - Ш§ЫҢШ¬Ш§ШҜ Щ…ЩҲШ¬ЩҲШҜЫҢШӘ
```

### Location Services
```
GET    /api/v1/locations/search  - Ш¬ШіШӘШ¬ЩҲЫҢ Щ…Ъ©Ш§ЩҶЫҢ
GET    /api/v1/locations/nearest - ЩҶШІШҜЫҢЪ©вҖҢШӘШұЫҢЩҶ Ш®ШҜЩ…Ш§ШӘ
GET    /api/v1/locations/route   - Щ…ШіЫҢШұЫҢШ§ШЁЫҢ
```

### Analytics
```
GET    /api/v1/analytics/stats   - ШўЩ…Ш§Шұ Ъ©Щ„ЫҢ
GET    /api/v1/analytics/report  - ЪҜШІШ§ШұШҙвҖҢЩҮШ§ЫҢ ШӘЩҒШөЫҢЩ„ЫҢ
```

### Backend
- **FastAPI** - Web Framework
- **SQLAlchemy** - ORM
- **Pydantic** - Data Validation
- **Transformers** - NLP Models
- **Geopy** - Location Services

### Frontend
- **React** - UI Framework
- **Leaflet** - Interactive Maps
- **Redux** - State Management
- **Tailwind CSS** - Styling

### Database
- **SQLite** - Development
- **PostgreSQL** - Production
- **Redis** - Caching
